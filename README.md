# FYP_SmartLearning

This is a web-based English learning platform developed as a course project.  
The application includes level assessment, learning modules, streak tracking, and user progress management.

## Tech Stack
- React + TypeScript (Vite)
- Firebase (Authentication & Firestore)
- Tailwind CSS + shadcn/ui
- React Router
- React Query

## Features
- User authentication
- English level assessment
- Learning modules (Grammar, Vocabulary, Reading, Writing, Listening, Speaking)
- Daily learning streak tracking
- User progress stored in Firebase Firestore
